Variables and data types

Variables

A variable is a word/identifier/object that hangs a value. The variables on python are defined dynamically, it means that it not neccesary to specify the type of data that will be store on them

The sympbol "=" is the assignment operator, for example

x = 1
text = "hola"

Data types in python

Here the main python's data types are described

Type Class Notes
int integer number Fixed precision variable, automatically converted to long type in overflow case
float decimal number double precision decimal number
bool boolean variable True or False (case-sensitive)
string string of characters a string literal is created writing it down between quotations marks ('_') or ("_")
String special characters

Not any symbol could use written directly in a string, there are several special characters that wait for special care to be included in a string of charcaters.

Due its nature, this character needs to be marked as unike or escaped. The backslash charcater (\) does this work, just appearing directly before the special charcater, e.g.
