Lists

The most basic data structure in Python is the sequence. Each element of a sequence is assigned a number (its position, or index). The first index is zero, the second index is one, and so forth. Python has six built-in types of sequences, but the most common ones are lists and tuples.

Lists are a datatype you can use to store a collection of different pieces of information as a sequence under a single variable name.

The list is a most versatile datatype available in Python, which can be written as a list of comma-separated values (items) between square brackets ([]) and ending with a semicolon.

var_list = [val1, val2, ..., valn];

Not all items in a list need to have the same type, e.g.:

list1 = ['physics', 'chemistry', 1997, 2000];

Creating empty list

Empty list could be created to be use in the future, the syntax is:

new_list_name = []

Creating list using range

Creating list of a sequence of numbers using range is as simple as:

my_list = range(51)
And you can use all the functionality of the range command.

Creating list using list comprehension

List comprehensions are a powerful way to generate lists using the for/in and if keywords we've learned. For example when we want to crate a list with some logic, as a list of all the even numbers from 0 to 50

evens_to_50 = [i for i in range(51) if i % 2 == 0]

Or the map of some function for certain values:

new_list = [f(i) for i in range(n)]

Creating a list by copying another

To create a copy of a list and assign it to a new one, breaking the link for prevent undesired changes you have to:

new_lst = original_lst[:]

Otherwise when you make any change in the original_lst it will be reflected in the new_list because they share direction in memory

Accessing Values in Lists

You can access an individual item on the list by its index. An index is like an address that identifies the item's place in the list. To access values in lists, use the square brackets for slicing along with the index or indices to obtain value available at that index, e.g.:

#obtaining the first item of list ListA
print ListA[0]

Built-in List Functions & Methods

Python includes following list functions:

len(list)

gives the number of items stored in the list

max(list)

returns item from the list with max value

min(list)

returns item from the list with min value

cmp(list1, list2)

compares elements of both lists

list(seq)

converts a tuple into list

Python includes following list methods

list.count(obj)

returns count of how many times obj occurs in list

list.index(obj)

returns the lowest index in list that obj appears. An error will occur if there is no such item.

list.append(obj)

appends object obj to list

list.insert(index, obj)

inserts object obj into list at offset index

list.extend(seq)

appends the contents of seq to list

list.remove(obj)

removes the first object obj from the list

list.pop(index = -1)

removes the item at index from the list and return it to you

del(list[idx])

removes the element living on idx position as .pop, but it won't return it

list.reverse()

reverses objects of list in place

list.sort(<func>)

sorts objects of list (optionally using function func to sort)

Concatenation (+)

[1,2,3] + [4,5,6] will produce [1,2,3,4,5,6]

Repetition (*)

["Hi"]*4 will produce ["Hi","Hi","Hi","Hi"]

Slice list_name[i:j]

obtains the portion of list_name starting with the index i and ending before the index j. If your list slice includes the very first or last item in a list (or a string), the index for that item doesn't have to be included, it is:

#slicing a list taking from the first to the n'th element
list_name[:n]

#slicing a list form the n'th to the last element
list_name[n:]
Slice with stride list_name[i:j:k]

obtains the portion of list_name starting with the index i (inclusive), ending on j index (exclusive), and considering k spaces between items in the sliced list.

Omitting indices

If you don't pass a particular index to the list slice, Python will pick a default. The default for the starting index is the first element of the list; the default for the ending index is the final element of the list; and the default for the stride index is 1.
